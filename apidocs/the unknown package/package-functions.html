recaptcha_get_html

public string recaptcha_get_html(string pubkey, string error, boolean use_ssl)

Gets the challenge HTML (javascript and non-javascript version). This is called from the browser, and the resulting reCAPTCHA HTML widget is embedded within the HTML form it was called from.

Parameters:
pubkey - A public key for reCAPTCHA
error - The error given by reCAPTCHA (optional, default is null)
use_ssl - Should the request be made over ssl? (optional, default is false)
Returns:
- The HTML to be embedded in the user's form.
